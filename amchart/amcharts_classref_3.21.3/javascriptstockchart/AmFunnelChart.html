Inheritance: AmFunnelChart → AmSlicedChart → AmChart

Extension for AmSlicedChart to create funnel/pyramid charts.

Example

var chart = AmCharts.makeChart("chartdiv",{
  "type"    : "funnel",
  "neckHeight"  : "30%",
  "neckWidth" : "40%",
  "titleField"  : "title",
  "valueField"  : "value",
  "dataProvider"  : [
    {
      "title": "Website visits",
      "value": 300
    },
    {
      "title": "Downloads",
      "value": 123
    }
  ]
});
